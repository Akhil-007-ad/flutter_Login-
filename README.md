## 📁 Getting Started

To run this project locally, follow these steps:

```bash
# Clone the repo
git clone https://github.com/Akhil-007-ad/flutter_Login-.git

# Move into the project folder
cd flutter_Login-

# Get Flutter packages
flutter pub get

# Run the app
flutter run
